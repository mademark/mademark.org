## Structure

- `/` — Homepage and overview
- `/marks/human-made` — Human Made mark definition (with JSON-LD)
- `/marks/human-designed-ai-made` — Human Designed, AI Made definition (with JSON-LD)
- `/marks/ai-made` — AI Made mark definition (with JSON-LD)
- `/governance` — Governance policy and stewardship

## Technical Implementation

- Static HTML with embedded JSON-LD in page headers
- Content negotiation for machine-readable responses
- Resolvable URIs at `mademark.org/ns`
- C2PA integration examples
