By format

Made Mark is designed to travel with every kind of generative output. Formats roll out in phases — what each one supports today:

Available now Text

Configure your AI to append the label automatically, or paste a generated label at the end of the document.

Set up → Available now Websites

Add JSON-LD to your page head and a badge to your footer. The generator outputs both, ready to paste.

Generate → Available now Images

Embed the label in a JPEG, PNG, or GIF's XMP metadata, with an optional visible badge — entirely in your browser.

Open the tool → Available now Vector

Embed the label in an SVG's own <metadata> element, with an optional vector badge — entirely in your browser.

Open the tool → Available now Video

Generate a labeled end card — 16:9, vertical, or square — plus a description snippet. The card carries the label in XMP.

Open the tool → Available now Audio

Embed the label in an MP3's ID3 tags — existing tags kept — plus an episode-notes snippet.

Open the tool → Available now Physical goods

Production vectors for engraving and embossing, a QR tag that links the object to its mark, and printable sticker sheets.
