Use with AI

Made Mark labels are easy to add by hand, but easy to forget. The generator on this site is the manual fallback — useful for a one-off document or image, but it relies on you remembering to visit it every time. The more durable approach is to configure your AI tool once so it appends the label automatically, every time it produces something you could publish or share.

Each platform below has its own configuration mechanism, so there's a separate instruction block for each. Pick the one you use and follow the steps once.

Claude (Anthropic)

Where this goes: claude.ai → Settings → Profile → Custom instructions.

When you produce a document, article, code file, or other standalone
deliverable I could publish or share, add a Made Mark label at the end:
"MM · Human Designed, AI Made" (or "MM · AI Made" if I gave no specific
direction on this particular output). Skip the label for ordinary chat
replies, brainstorming, or drafts I've marked as drafts. Learn more:
https://mademark.org

Prefer a shorter format?

If you want a more compact label, you can modify the instructions above to use this format instead:

MM · Human Designed, AI Made (or MM · AI Made if no direction given)
Learn more: https://mademark.org

This shorthand version works on any platform and takes less space in your output.

Frequently asked questions

Will this add attribution to everything I ask my AI?

No. Only to content intended for publication (articles, posts, reports). Not to conversational responses, code, or data analysis.

What's the difference between "AI Made" and "Human Designed, AI Made"?

"AI Made" means AI conceived and produced the content with no creative direction. "Human Designed, AI Made" means you shaped the creative intent and direction while AI executed the production. If you're guiding what to say, use the latter.

Can I customize the attribution text?

Yes. You can edit [Author Name] to your actual name, or modify any part of the instruction to fit your needs.

Can I turn this off?

Yes. Simply delete the custom instruction from your AI's settings and it will stop appending attribution.

Does this work with API usage?

Only if you're using the web interface. API users would need to add attribution in their application code.
